## Objective
Prove that an existing scenario re-aggregates and reflects the new exposure after a data source is added.

## Preconditions
- A portfolio with an existing scenario that already shows values exists.

## Scenario: new data source changes scenario values
```gherkin
Given a portfolio with an existing scenario showing values
When a new data source is uploaded and processed
Then the scenario values are adapted to include the new data
```
