## Objective
Prove the date shortcut buttons set both "Policy live at" and "Exposure run off date" to the expected value.

## Preconditions
- A scenario list is open on the Aggregations page.

## Scenario: a date shortcut sets both dates
```gherkin
Given a scenario list is open
When the user clicks "<button>"
Then "Policy live at" and "Exposure run off date" are both set to <date>
```

### Examples
| button | date |
|--------|------|
| Previous 1/1 | 1 January of the year before the current year |
| Today | today's date |
