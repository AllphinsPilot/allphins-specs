## Objective
Prove that changing which policies are selected and re-computing updates the scenario exposures accordingly.

## Preconditions
- A portfolio with several policies and a computed scenario exists.

## Scenario: selecting different policies changes computed exposure
```gherkin
Given a scenario on a portfolio with several policies
When the user selects a different set of policies and clicks Compute
Then the scenario exposures are adapted to the selected policies
```
